# todolsit-project
电子科大——小米训练营的结营项目
一个todolist快应用，使用快应用官方IDE编写，详见https://www.quickapp.cn/
快应用命名为快简记，上线至小米应用商店。

主要功能如下
快简记备忘录(包括待办事项和笔记)

基础功能
• 使用 kaldi 进行语音输入待办事项 ；
• 支持修改待办事项 ；
• 给待办事项添加完成时间。

额外实现功能
• 样式优化、动态效果 ； 
• 从手机本地存储、读取数据 ；
• 到 deadline 了进行震动提醒 ；
• 增加一了个简单统计页面，对待办事项进行统计；
• 状态从 todo/done 变成 todo/doing/done。
状态可以自动更新，并增加了超时的状态

• 增加笔记页面（支持kaldi 进行语音输入）；
• 可以对事件的重要程度进行标记
• 支持一键删除笔记以及待办事项
• 可以手动将已完成的待办事项划分到done状态
